# odpod-site

Published build output for **OdPod / Obiter Dictum** — a static site that explores a
Swedish pop-culture podcast (search, tag graph, timeline, image gallery).

> ⚙️ **This repo contains only publishing artifacts.** It is generated and overwritten by a
> publish script — no development happens here. All source lives in a separate (private)
> `odpod` repo and is built and pushed from there.

**Live site:** https://bobbytable.github.io/odpod-site/
